## React Frontend für ein 'Stein, Schere, Papier' Spiel

1. Voraussetzungen für das Ausführen in der IDE: NodeJS mit npm muss installiert sein
2. Zum Ausführen der App müssen diese Scripte ausgeführt werden

### `npm install`

Installiert alle notwendigen Packete lokal

### `npm start`

Die App läuft im develop Mode unter.\
[http://localhost:3000](http://localhost:3000) im Browser.


### `npm run build`

Baut die App local
